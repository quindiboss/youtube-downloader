# PROJOPP_AP_e7 - Skrypt prezentacji (1 minuta)

## 🎯 CEL PROJEKTU (10 sek)
YouTube Downloader - aplikacja webowa do pobierania filmów z YouTube
w różnych formatach: MP4 (720p, 1080p, najlepsza jakość), MP3 i M4A.

## 🧪 ETAP 7 - TESTOWANIE (10 sek)
- Dodano testy jednostkowe dla backendowego modułu `server/utils.js`.
- Sprawdzono walidację linków YouTube i budowanie argumentów `yt-dlp`.
- Testy uruchamia się poleceniem `npm test`.

## 🖥️ INTERFEJS (10 sek)
- Pole wyboru formatu: MP4 / MP3 / M4A
- Przyciski jakości wideo dla MP4: 720p / 1080p / najlepsza jakość
- Pasek postępu pobierania w czasie rzeczywistym

## ⚡ FUNKCJONALNOŚCI (15 sek)
1. Wklejasz link YouTube
2. Wybierasz format albo jakość wideo
3. Klikasz "Pobierz"
4. Widzisz postęp pobierania
5. Pobierasz gotowy plik

## 📄 DOKUMENTACJA I WERSJE (5 sek)
- README.md z pełną dokumentacją
- Testy: `server/utils.test.js`
- Wersja: 0.7 (Etap 7 ukończony)
- Repozytorium: GitHub
