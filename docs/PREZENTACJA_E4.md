# PROJOPP_AP_e4 - Skrypt prezentacji (1 minuta)

## 🎯 CEL PROJEKTU (10 sek)
YouTube Downloader - aplikacja webowa do pobierania filmów z YouTube 
w różnych formatach: MP4 (720p, 1080p, najlepsza jakość), MP3 i M4A.

## 🛠️ UŻYTE TECHNOLOGIE (10 sek)
- **Frontend:** React 18 + Vite
- **Backend:** Node.js + Express
- **Pobieranie:** yt-dlp + FFmpeg
- **Wersjonowanie:** Git + GitHub

## 🖥️ INTERFEJS (10 sek)
- Minimalistyczny design (glassmorphism)
- Pole na link YouTube
- Lista rozwijana z formatami
- Pasek postępu w czasie rzeczywistym
- Przycisk pobierania pliku

## ⚡ FUNKCJONALNOŚCI (15 sek)
1. Wklejasz link YouTube
2. Wybierasz format (MP4/MP3/M4A)
3. Klikasz "Pobierz"
4. Widzisz postęp pobierania na żywo
5. Pobierasz gotowy plik na dysk

## 🚀 KIERUNKI ROZWOJU (10 sek)
- Obsługa playlist
- Więcej serwisów (Vimeo, TikTok)
- Historia pobrań
- Wersja mobilna (PWA)

## 📄 DOKUMENTACJA I WERSJE (5 sek)
- README.md z pełną dokumentacją
- Wersja: 0.3 (Etap 3 ukończony)
- Repozytorium: GitHub
